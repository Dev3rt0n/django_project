Sobre o Projeto


Este projeto é todo desenvolvido com Django (rodando na versão 2.1.7), framework de Python (rodando na versão 3.7.2), atuando no backend. Outras tecnologias incluem bootstrap e javascript no frontend.

Como uma metalinguistica, o projeto pretende explicar o funcionamento do django utilizando o próprio Django para isto.

A mente por trás deste humilde projeto sou eu, Everton Oliveira, estudante de análise de sistemas pela FAETERJ-RIO.

Busco, como objetivo, apenas por em prática meus estudos sobre este maravilhoso framework

A história do Django


Na imagem ao lado é possivel ver a primeira aplicação web feita em Django.

Brincadeira!! O Django não é nem um pouco velho, sua primeira versão foi lançada em Julho de 2005. Seu nome tem como inspiração o grande guitarrista de jazz Django Reinhardt, que influênciou muitos outros músicos como Tony Iommi, guitarrista da banda Black Sabbath.

Tudo começou quando dois programadores de um jornal local da cidade de Kansas, nos Estados Unidos, resolveram criar um framework de python que lhes facilitassem o trabalho de administrar uma pagina web tão dinâmica como de um jornal. A intenção também era criar uma ferramenta de fácil manipulação para que jornalistas não programadores pudessem utiliza-la.

Estes programadores são respectivamente Adrian Holovaty e Simon Willison que ganharam inumeros premios por esta criação.
