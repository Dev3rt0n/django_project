Sobre o Projeto

Este projeto é todo desenvolvido com Django, framework de Python, atuando no backend. Outras tecnologias incluem bootstrap e javascript no frontend.

Como uma metalinguistica, o projeto pretende explicar o funcionamento do django utilizando o próprio Django para isto.

O projeto é todo desenvolvido por mim, Everton oliveira, estudante de análise de sistemas pela FAETERJ-RIO.
